    Introduce

    TriImageDB의 모든 이미지는 고유한 UID를 가지고 있습니다

    Target과 UID로 특정 이미지를 읽어올 수 있습니다

    curl "https://trinets.xyz:8443/static/<target>/<uid>.<png/jpg...>"

    <img src="https://trinets.xyz:8443/static/<target>/<uid>.<png/jpg...>">

    document.getElementById('image').src = 'https://trinets.xyz:8443/static/<target>/<uid>.<png/jpg...>'

    TriNet의 매우 딥 다크한 이미지 데이터베이스, TriImageDB에 오신것을 환영합니다!
    이곳에 일반인들이 올 경우는 매우 희박하겠지만, 어찌됬든 환영합니다!

    TriImageDB에서는 다음의 사이트들에서 이미지를 수집합니다
    Pixiv, Danbooru, Gelbooru, Rule 34, Pinterest

    Querys

    target

    TriImageDB에서는 다음의 단계로 이미지를 분류합니다

    Target 값이 없을 경우 기본값 neko를 사용합니다

    curl "https://trinets.xyz:8443/api/v1?target=neko"
curl "https://trinets.xyz:8443/api/v1?target=neko-r19"
curl "https://trinets.xyz:8443/api/v1?target=hentai"
curl "https://trinets.xyz:8443/api/v1?target=hentai-gif"

    <iframe src="https://trinets.xyz:8443/api/v1?target=neko">
<iframe src="https://trinets.xyz:8443/api/v1?target=neko-r19">
<iframe src="https://trinets.xyz:8443/api/v1?target=hentai">
<iframe src="https://trinets.xyz:8443/api/v1?target=hentai-gif">

    document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?target=neko'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?target=neko-r19'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?target=hentai'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?target=hentai-gif'
    Target 설명 NSFW(19금)
    neko 네코(고양이 수인) 일러스트를 제공합니다 아니요
    neko-r19 네코(고양이 수인) 야짤을 제공합니다 네
    hentai 광범위한 야짤을 제공합니다 네
    hentai-gif 광범위한 야움짤을 제공합니다 네
    머신러닝을 통해 야짤과 일반 일러스트를 구분합니다

    type

    TriImageDB에서는 다음과 같은 데이터 형식을 제공합니다

    Type 값이 없을 경우 기본값 page를 사용합니다

    curl "https://trinets.xyz:8443/api/v1?type=url"
curl "https://trinets.xyz:8443/api/v1?type=page"
curl "https://trinets.xyz:8443/api/v1?type=list"
curl "https://trinets.xyz:8443/api/v1?type=buffer"
curl "https://trinets.xyz:8443/api/v1?type=listImg"
curl "https://trinets.xyz:8443/api/v1?type=redirect"

    <iframe src="https://trinets.xyz:8443/api/v1?type=url">
<iframe src="https://trinets.xyz:8443/api/v1?type=page">
<iframe src="https://trinets.xyz:8443/api/v1?type=list">
<iframe src="https://trinets.xyz:8443/api/v1?type=buffer">
<iframe src="https://trinets.xyz:8443/api/v1?type=listImg">
<iframe src="https://trinets.xyz:8443/api/v1?type=redirect">

    document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?type=url'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?type=page'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?type=list'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?type=buffer'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?type=listImg'
document.getElementById('iframe').src = 'https://trinets.xyz:8443/api/v1?type=redirect'
    Type 설명 제공 방식
    url 렌덤으로 한 이미지의 URL을 제공합니다 Raw Text
    page 렌덤으로 한 이미지를 읽습니다 HTML
    list DB에 존제하는 모든 이미지의 UID를 제공합니다 JSON
    buffer 렌덤으로 한 이미지의 바이너리를 제공합니다 Binary
    listImg DB에 존제하는 모든 이미지를 읽습니다 HTML
    redirect 렌덤으로 한 이미지의 URL로 리다이렉트 됩니다 HTTP 302
    listImg는 로딩시 매우 오랜 시간이 걸릴 수 있습니다

    Errors

    TriImageDB의 오류코드 목록들입니다:

    오류코드 뜻
    404 Not Found -- API 버전이 맞지 않음 / 없는 이미지 UID / Query가 잘못됨
    423 Locked -- 해당되는 Target에 아무런 이미지도 없음 / 아직 공개되지 않은 API
    500 Internal Server Error -- 서버에 문제가 있음
    503 Service Unavailable -- 서버 업데이트를 위해 잠시 서버가 다운됨
